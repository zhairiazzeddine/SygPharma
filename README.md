# SygPharma – نظام إدارة صيدليات الحراسة والصيدليات المفتوحة

منصّة ويب لمتابعة صيدليات الحراسة والصيدليات المفتوحة في مدينة واحدة (حاليًا: سيدي يحيى الغرب)،  
مبنية باستخدام **Laravel 12 + Filament v5 + MySQL + Leaflet (OpenStreetMap)**.

المشروع يوفّر:

- واجهة عامة للزوار لمعرفة:
  - الصيدليات المفتوحة حاليًا بالتوقيت العادي.
  - صيدليات الحراسة (الليل، عطلة نهاية الأسبوع، الأعياد والعطل الرسمية).
  - أقرب صيدلية للزائر مع توجيه مباشر عبر خرائط Google.
- لوحة تحكم للمسؤول (Super Admin) لإدارة:
  - الصيدليات، الأحياء، أوقات العمل، المناوبات، الاستثناءات، المستخدمين…
- لوحة تحكم للصيدلاني لإدارة:
  - بيانات الصيدلية، ساعات العمل، المناوبات، وبعض الإحصائيات الخاصة به.

## 🧩 المزايا الرئيسية

### واجهة الزوار (Public Website)

- عرض **الصيدليات المفتوحة حاليًا** (خدمة نهارية) بناءً على:
  - اليوم الحالي
  - ساعات العمل المحددة لكل صيدلية
  - الاستثناءات (إغلاق خاص، عطل…)
- عرض **صيدليات الحراسة لليوم الحالي**:
  - مناورات الليل
  - عطلة نهاية الأسبوع (السبت/الأحد)
  - الأعياد والعطل الرسمية
- خريطة تفاعلية باستخدام **Leaflet + OpenStreetMap**:
  - عرض الصيدليات على الخريطة مع Marker لكل صيدلية.
  - إظهار تفاصيل الصيدلية (اسم، حي، عنوان، هاتف).
- زر **"أقرب صيدلية لي الآن"**:
  - استخدام Geolocation من المتصفح لحساب أقرب صيدلية.
  - تمييز الكارد الأقرب وتحريك الشاشة إليها.
- زر **"الموقع عبر الخريطة"** لكل صيدلية:
  - فتح **Google Maps Directions** مع الوجهة جاهزة (الصيدلية + المدينة + الإحداثيات).

### لوحة المسؤول (Admin Panel)

مبنية باستخدام **Filament v5**:

- إدارة **الصيدليات (Pharmacies)**:
  - الاسم، الحي (Area)، العنوان، الهاتف، إحداثيات الخريطة، حالة التفعيل.
- إدارة **الأحياء (Areas)**.
- إدارة **ساعات العمل (Opening Hours)** لكل صيدلية:
  - اليوم (0–6 / الأحد – السبت).
  - أوقات الفتح والإغلاق (نظام 24 ساعة).
  - إمكانية تحديد اليوم كـ "مغلق".
- إدارة **المناوبات (Duties)**:
  - ربط المناوبة بصيدلية محددة.
  - تاريخ ووقت البداية والنهاية.
  - نوع المناوبة:
    - `night` – مناوبة ليلية.
    - `weekend` – عطلة نهاية الأسبوع.
    - `holiday` – الأعياد والعطل الرسمية.
  - التحقق من صحة الفترة (لا فترات معكوسة، لا تداخل غير منطقي، إلخ).
- إدارة **الاستثناءات (Opening Exceptions)** (إغلاق خاص/استثنائي).
- إدارة **المستخدمين (Users)**:
  - الأدوار:
    - `super_admin` – مسؤول عام للنظام.
    - `pharmacist` – صيدلاني مرتبط بصيدلية (أو أكثر).
- إحصائيات (Dashboard) للمسؤول:
  - عدد الصيدليات الفعّالة.
  - عدد المناوبات لآخر 30 يومًا.
  - توزيع المناوبات حسب الأحياء (Doughnut / Pie Chart).
  - قائمة الصيدليات الأكثر مناوبة في آخر 30 يومًا.

### لوحة الصيدلاني (Pharmacist Panel)

- إدارة بيانات الصيدلية الخاصة به:
  - الاسم، العنوان، الحي، الهاتف، الإحداثيات…
- إدارة **ساعات العمل** لصيدليته فقط.
- إدارة **المناوبات** الخاصة بصيدليته (مع منع التداخل).
- صفحة **"حسابي"** لتعديل:
  - الاسم، البريد الإلكتروني، كلمة السر (مع تأكيد كلمة السر).
- إحصائيات خاصة بالصيدلاني:
  - عدد المناوبات لآخر 30 يومًا.
  - Line Chart لتطور المناوبات حسب النوع (ليل/عطل/ويكند) في آخر 30 يومًا.

## 🏗️ التكديس التقني (Tech Stack)

- **Backend**: Laravel 12 (PHP 8.2+)
- **Admin Panel**: Filament v5
- **Database**: MySQL 8
- **Frontend**:
  - Blade + Bootstrap (قالب MediTrust)
  - Leaflet.js + OpenStreetMap
  - JavaScript مخصص للـ:
    - Google Maps Directions
    - Geolocation + أقرب صيدلية
    - عدّاد الوقت المتبقي للمناوبة
- **Deployment**:
  - استضافة مشتركة (مثال: Nindohost)
  - Document root موجه إلى `public_html` مع Laravel في مجلد فرعي

## 👥 الأدوار (Roles)

- `super_admin`
  - وصول كامل للوحة المسؤول (Backoffice).
  - إدارة الصيدليات، الأحياء، المستخدمين، المناوبات، الاستثناءات، إلخ.
  - **حساب مالك النظام (Owner)** محمي من الحذف والتعديل من طرف الآخرين.

- `pharmacist`
  - وصول إلى لوحة الصيدلاني.
  - إدارة صيدليته (أو صيدلياته) فقط:
    - بيانات الصيدلية
    - ساعات العمل
    - المناوبات
  - تعديل بيانات حسابه الخاص من صفحة "حسابي".

## 🧬 بنية بعض الـ Models

- `Area`
  - `id`, `name`, `slug`, …
  - علاقة: `hasMany(Pharmacy::class)`

- `Pharmacy`
  - `id`, `area_id`, `name`, `address`, `phone`, `latitude`, `longitude`, `is_active`, …
  - علاقات:
    - `belongsTo(Area::class)`
    - `hasMany(Duty::class)`
    - `hasMany(OpeningHour::class)`
    - `hasMany(OpeningException::class)`
    - `hasMany(User::class)` (صيادلة مرتبطون بنفس الصيدلية)

- `Duty`
  - `id`, `pharmacy_id`, `starts_at`, `ends_at`, `duty_type`, `created_by`, …
  - `casts`: `starts_at`, `ends_at` → `datetime`
  - Scope: `scopeActive()` لفلترة المناوبات النشطة في وقت معين.

- `OpeningHour`
  - `id`, `pharmacy_id`, `day_of_week`, `opens_at`, `closes_at`, `is_closed`, …

- `User`
  - `id`, `name`, `email`, `password`, `role`, `pharmacy_id`, …
  - Helpers:
    - `isSuperAdmin()`
    - `isPharmacist()`
    - `isOwner()` (مالك النظام)
